# 加油站自助柜员机


## 系统简介

加油站自助充值、发卡设备的操作终端，可以插入纸币、自动发放射频卡。

后台与结算系统对接。



## 技术介绍

1、采用WPF C# 开发，使用 MVVM模式

2、自定义数据控件模板和样式，自定义按钮控件

3、后台与钱箱硬件通讯、卡箱设备通讯。
